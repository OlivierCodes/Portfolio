Application mobile IA

Cette application a pour but de détecter le sexe d'une personne, que ce soit masculin ou féminin. J'ai développé cette application pour répondre aux besoins spécifiques d'un client.

L'objectif principal était de fournir une solution innovante pour aider les utilisateurs à identifier le sexe avec précision.

Grâce à l'interface utilisateur conviviale que j'ai intégrée, le client peut facilement utiliser cette application sans nécessiter de connaissances techniques avancées.

Ce projet d'application mobile, créé avec Xamarin et utilisant des fonctionnalités d'intelligence artificielle, a été très apprécié par le client pour sa fonctionnalité et sa simplicité d'utilisation.

Technologies utilisées: XAMARIN, C#, AZURE.
